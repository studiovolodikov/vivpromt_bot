# Инструкция по публикации на GitHub  
  
## 1. Создайте приватный репозиторий на GitHub  
  
1. Перейдите на https://github.com/new  
2. Имя: prompt-strategist-bot  
3. Выберите Private  
4. НЕ инициализируйте с README  
5. Create repository  
  
## 2. Свяжите с GitHub  
  
```bash  
git remote add origin https://github.com/ВАШ_НИК/prompt-strategist-bot.git  
git branch -M main  
git push -u origin main  
```
